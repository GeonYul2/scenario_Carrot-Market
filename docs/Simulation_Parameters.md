# [Simulation] AI 가상 유저 행동 및 실험 파라미터 명세

## 1. 실험 설계 (Experimental Design)
- **대상**: 당근알바 고관여 미지원 세그먼트 (N=50,000)
- **실험군(Treatment)**: 개인화 푸시 메시지 발송 (90%)
- **대조군(Control/Holdout)**: 아무런 조치 없음 (10%)
- **분기 로직**: `user_id` 기반 Hash 분기 (실험 오염 방지)

## 2. AI 유저 행동 확률 설정 (Ground Truth)
실험의 인과관계를 입증하기 위해 AI 가상 유저의 행동 확률을 다음과 같이 정의한다.

| 유저 행동 구분 | 확률값 ($P$) | 비고 |
| :--- | :--- | :--- |
| **자연 지원 확률 (Baseline)** | $2.0\%$ | 대조군의 기본 전환율 |
| **푸시 수신 후 지원 확률** | $5.5\%$ | 메시지 노출 시 기대 전환율 (LIFT +175%) |
| **푸시 수신 후 알림 차단율** | $0.4\%$ | 가드레일 지표 (임계치 0.5% 미만 관리) |

## 3. 통계적 검증 기준
- **유의 수준 ($\alpha$)**: $0.05$
- **검정력 ($1-\beta$)**: $0.80$
- **최소 효과 크기 (MDE)**: $1.0\%p$ 상승 시 유의미한 캠페인으로 간주
